# Решение: выбор ниши для side-project

**Дата:** 2025-01-16

## Контекст
- Основная работа: Nebius Academy (CCO)
- Цель: side-project → замена основной работы
- Ресурсы: мало времени, есть сильный разработчик

## Рассмотренные варианты

| Ниша | Плюсы | Минусы |
|------|-------|--------|
| AI-агенты для Real Estate Дубай | Высокие чеки | Нет контактов в нише |
| AI для визового бизнеса MENA | Чеки $6-25k | Много compliance |
| Консалтинг по лидгену | Знаешь тему | Не скейлится |
| AI-автоматизация для B2B Sales | Экспертиза изнутри, нетворк | — |

## Решение

**Фокус: AI-автоматизация для B2B Sales команд**

Причины:
1. Знаешь изнутри — сам строил, сам болел
2. Есть нетворк (ex-Russian стартапы, Nebius)
3. Есть разработчик
4. Первый клиент = из своего круга
5. Вырастает в продукт, не консалтинг

## Следующий шаг

Взять 1-2 знакомых фаундера, сделать бесплатно/дешево автоматизацию outbound (Apollo + enrichment + AI-персонализация), получить кейс.
