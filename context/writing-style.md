# Writing Style Guide

## Email Style

### Characteristics
- Short paragraphs (1-3 sentences)
- Clear call-to-action
- No unnecessary greetings
- Professional but warm

### Example (English)
```
Hi [Name],

Quick question about [topic].

[1-2 sentences of context]

[Clear question or request]

Best,
Pasha
```

### Example (Russian)
```
Привет [Name],

[Суть вопроса в 1-2 предложениях]

[Конкретный вопрос или просьба]

Паша
```

## Message Style (Telegram/Slack)

### Characteristics
- Very brief
- No greetings for ongoing conversations
- Emoji sparingly (if at all)
- Direct to the point

### Example
```
посмотри это - [link]
что думаешь?
```

## Document Style

### Characteristics
- Headers for structure
- Bullet points > paragraphs
- Tables for comparisons
- Code blocks for technical

### Structure
```
# Title

## Context
[Brief background]

## Problem
[What needs solving]

## Options
1. Option A - pros/cons
2. Option B - pros/cons

## Recommendation
[Clear recommendation with reasoning]

## Next Steps
- [ ] Action 1
- [ ] Action 2
```

## Social Media Style

### LinkedIn
- Professional but personable
- Stories over statements
- Data to support claims
- Clear takeaway

### Telegram Channel
- More casual
- Mix of Russian/English
- Direct opinions
- Quick insights

## Avoid

- Em-dashes (—) - use regular dash
- "I think" / "I believe" - just state it
- Excessive hedging
- Long introductions
- "Hope this helps!"
- "Let me know if you have questions"

## Tone by Context

| Context | Tone |
|---------|------|
| Sales email | Professional, confident |
| Team message | Casual, direct |
| Technical doc | Precise, structured |
| Social post | Personal, engaging |
| Support | Helpful, patient |
