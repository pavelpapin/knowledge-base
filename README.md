# Pasha's Knowledge Base

Репозиторий для синхронизации контекста между Claude Code сессиями.

## Структура

```
/CLAUDE.md      — профиль, предпочтения, контекст
/decisions/     — принятые решения с обоснованием
/projects/      — текущие проекты и их статус
/notes/         — заметки, идеи, research
```

## Как использовать

Claude автоматически читает CLAUDE.md при старте сессии.
Важные решения фиксируются в /decisions/ с датой.
Проекты трекаются в /projects/.
